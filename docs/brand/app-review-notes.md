# App Review — notes for the reviewer

Paste into **App Store Connect → your version → App Review Information → Notes**.

---

Expexp is an iPhone film double-exposure camera. All processing is on-device; the app has no
account, no login, and collects no data.

**Please test on a physical iPhone — the camera does not function in the Simulator.**

How to try it:
1. Launch the app and allow **Camera** and **Photo Library** access when prompted.
2. Tap the round shutter button to take a photo, then tap it again 2–3 more times, aiming at
   slightly different things each time. A translucent "ghost" of the previous shots shows through.
3. Tap the green checkmark (✓) to save. The blended multiple-exposure photo is written to the
   Photo Library and a thumbnail appears bottom-left (tap it to open Photos).
4. Optional: the bottom-left label toggles **Raw / Edge Lock / Face Lock** (exposure alignment);
   the **⋯** button switches the visual "look."

No demo account is required.
